# Task 1: To-Do List Application in Python

## Project Overview

The To-Do List Application is a simple command-line based project developed using Python. This application allows users to manage their daily tasks efficiently by adding, viewing, marking, and deleting tasks.

The project stores tasks permanently using a JSON file, which helps maintain data even after the program is closed. It is a beginner-friendly project that demonstrates file handling, functions, loops, conditional statements, and error handling in Python.

---

# Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Store tasks permanently using JSON file
* User-friendly menu-driven interface
* Basic error handling implemented
* Displays task completion status

# How the Program Works

1. The program displays a menu with different options.
2. Users can:

   * Add a task
   * View all tasks
   * Mark tasks as completed
   * Delete tasks
   * Exit the application
3. Tasks are stored in a JSON file (`tasks.json`).
4. The application updates the file whenever changes are made.
5. Error handling prevents invalid task operations.
